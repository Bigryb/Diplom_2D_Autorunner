## [1.0.0] - 2026-06-18

### Added

- Выпущен первый стабильный релиз игры **RusRunner**.
- Добавлены готовые сборки игры для Windows, macOS и Linux в GitHub Releases.
- Добавлен Unity-проект `RusRunner` с основной игровой сценой `RunnerScene.unity`.
- Реализован линейный 2D-автораннер с короткими игровыми сессиями.
- Добавлено главное меню, экран настроек, HUD и экран поражения.
- Реализовано управление персонажем:
  - обычный прыжок;
  - дополнительный прыжок в воздухе;
  - подкат;
  - повторный запуск после поражения.
- Добавлены препятствия разных типов:
  - наземные препятствия;
  - воздушные препятствия;
  - высокие препятствия, требующие подката.
- Реализованы бонусы временного действия:
  - щит;
  - ускорение;
  - удвоение очков.
- Добавлена система очков, дистанции, скорости и сохранения лучшего результата.
- Реализован постепенный рост скорости и сложности игрового процесса.
- Добавлена генерация бесконечной трассы.
- Реализовано повторное использование игровых объектов через object pooling.
- Добавлено локальное сохранение настроек и рекорда через `PlayerPrefs`.
- Добавлены настройки громкости музыки, громкости звуковых эффектов и целевой частоты кадров: 60, 120 и 180 FPS.
- Добавлены графические и звуковые ресурсы проекта в каталог `Resources`.
- Добавлена пояснительная записка дипломного проекта `ДИПЛОМ.docx`.
- Добавлена лицензия MIT.

### Technical

- Проект подготовлен для Unity `6000.0.43f1`.
- Используется C# для реализации игровой логики и модульного ядра.
- Используется Unity 2D Physics для обработки движения, прыжка, подката и столкновений.
- Используется Universal Render Pipeline `17.0.4` для рендеринга 2D-сцены.
- Используется `uGUI 2.0.0` для главного меню, настроек, HUD и экрана Game Over.
- Подключён пакет Input System `1.13.1` для дальнейшего развития системы ввода.
- Игровая логика разделена на самостоятельные модули:
  - `RunnerBootstrap` — координация игрового цикла;
  - `GameContext` — хранение состояния забега;
  - `ProgressionSystem` — рост скорости и сложности;
  - `ScoreSystem` — очки, дистанция и рекорд;
  - `PowerUpSystem` — временные бонусы;
  - `RunnerController` — управление и физика персонажа;
  - `TrackGenerator` — генерация трассы, препятствий и бонусов.
